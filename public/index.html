    About

    This tool lets you find which GP practice catchment areas cover a given location. Search by postcode or click on the map to see which GP practices serve that area.

    The list of GP practices comes from NHS Digital (the 'epraccur.csv' file on that page) and catchment data comes from the GP Catchments Data Download Hub . Data is updated automatically every week.

    This page was created by Robin Wilson, a freelance geospatial software engineer.
